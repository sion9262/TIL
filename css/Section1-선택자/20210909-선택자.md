# 선택자

### CSS 선택자
- CSS 선택자는 적용될 요소를 선택한다.
- 주요 선택자는 - Type, Class, ID가 있다.

### Type Selector
- 하나의 HTML에 있는 모든 해당 요소를 선택한다.
- 하나의 전체 웹페이지에서 일관성 있게 스타일링을 할 때 사용한다.

```css
h2 {
    color : purple;
}
```

### ID Selector

- `#idName`과 같이 `#`을 붙혀 사용된다.
- `id`를 가진 요소는 문서에 하나만 존재해야한다.

```css
#welcom-title {
    color : red;
}
```

### Class Selector
- `.ClassName`과 같이 `.`을 붙혀 사용된다.
- `Class`를 갖는 모든 요소를 선택한다.
- `Class`는 스페이싱을 통해 여러 클래스를 가질 수 있다. 이 경우 Cascading원칙에 따라 뒤에 작성된 스타일이 적용된다.

```css
.blue {
    color : blue;
}

.red {
    color : red;
}
```

### Attribute Selector

#### [attr], [attr=value], [attr^]. [attr$], [attr_]

- 특정 속성을 가진 요소를 선택한다.
- 주로 `input` 태그에 많이 사용된다.
- `^`를 사용하면 해당 value로 시작하는 요소를 선택한다.
- `$`를 사용하면 해당 value로 끝나는 요소를 선택한다.
- `*`를 사용하면 해당 value를 포함하는 요소를 선택한다.
```css
/* [attr] 형식 */
a[target] {
    color : hotpink;
}
/* [attr=value] 형식 */
a[href="https://example.org"] {
    color : indigo;
}

input[type="submit"] {
    background-color : green;
}

/* [attr^=value]*/
a[href^="http://"]{
    color : red;
}
/* [attr$=value]*/
a[href$=".com"] {
    color : silver;
}
/* [attr*=value]*/
a[href*="example"] {
    color : sienna;
}
```
### Pseudo-Class Selector (가상 클래스 선택자)

- 실제로 존재하지 않는 상태를 가상 클래스로 만들어 해당 클래스를 선택한다.
- 다음과 같은 형식으로 작성된다.

```css
selector:____{
    property : value;
}
```

- `first-child`는 형제 요소 중 첫번째 요소를 선택한다.
```css
/* first-child */
li:first-child {
    color : green;
}
```

- `last-child`는 형제 요소 중 마지막 요소를 선택한다.
```css
/* last-child */
li:last-child{
    color : pink;
}
```

- `nth-child()` 는 형제 요소 중 N번째 해당되는 요소를 선택한다.
```css
/* nth-child*/
li:nth-child(2){
    color : red;
}
li:nth-child(2n){
    color : blue;
}
li:nth-child(odd){
    color : black;
}
```

- `first-of-type`는 형제 요소 중 자신과 일치한 첫번째 요소를 선택한다.
```css
/* first-of-type */
li:first-of-type{
    color : pink;
}
```
- `last-of-type`는 형제 요소 중 자신과 일치한 마지막 요소를 선택한다.
```css
/* last-of-type */
li:last-of-type{
    color : black;
}
```
- `nth-of-type()`는 형제 요소 중 자신과 일치한 N번째 요소를 선택한다.
```css
/* nth-of-type() */
li:nth-of-type(2){
    color : red;
}
```
- `not()`은 selector 요소를 고르 되 not 안의 selector가 아닌 요소를 선택한다.
```css
/* input 중 type이 password 가 아닌 요소*/
input:not([type=password]){
    background-color : gray;
}
/* input 중 pw class가 아닌 요소 */
input:not(.pw){
    background-color : red;
}
```
- `link`는 링크를 한번도 방문한 적 없을 때 
  - 하이퍼링크 상태에 따라 변함.
```css
a:link {
    color : pink;
}
```
- `visited`는 링크를 한번이라도 방문한 적 있을 때
  - 하이퍼링크 상태에 따라 변함.
```css
a:visited{
    color : tomato;
}
```
- `hover` 마우스를 요소에 올려놨을 때 적용되는 상태이다.
```css
input[type=button]:hover{
    background-color : red;
}
```
- `active` 마우스를 클릭 후 떼기 전까지의 상태이다.
```css
input[type=button]:active{
    background-color : white;
}
```
- `focus` 요소가 포커스 받는 경우, 사용자가 요소를 클릭하거나 TAP할 경우 적용되는 상태이다.
```css
input[type=text]:focus{
    background-color: blue;
}
```
- 우선 순위는 LVHA 순서이다. `:link` -> `:visited` -> `:hover` -> `active`


- `enabled`는 해당 요소의 속성이 enabled 일때 적용되는 상태이다.
```css
input[type=text]:enabled {
    background-color: skyblue;
}
```
- `disabled`는 해당 요소의 속성이 disabled 일때 적용되는 상태이다.
```css
input[type=text]:disabled{
    background-color: silver;
}
```
- `checked` `input radio` 요소 등 속성이 `checked`인 경우 적용되는 상태이다.
```css
input[type=radio]:checked{
    outline: 3px solid red;
}
input[type=checkbox]:checked{
    outline: 3px solid blue;
}
```

### Pseudo-Element Selector 가상 요소 선택자
- CSS3에서는 가상 클래스와 요소 선택자를 구분하기 위해 `selector::__`로 가상 요소 선택자를 쓴다.
- 가상 클래스 선택자는 상태에 따라 변화하는 것이지만, 가상 요소 선택자는 실제로 존재하지 않는 요소나 범위를 만들어 적용한다.
- 가상 요소 선택자는 복사가 되지 않는다.

- `before` 선택한 요소 앞에 가상의 인라인 요소를 생성한다. `content` 로 글자를 넣을 수 있음.
```css
.movie::before{
    content: 'MOVIE';
    color : indianred;
}
```
- `after` 선택한 요소 뒤에 가상의 인라인 요소를 생성한다. `content` 로 글자를 넣을 수 있음. 
```css
.movie::after{
    content: "FAVORITE";
    color : blue;
}
```
- `first-letter` 첫번째 글자에 대한 스타일링을 적용한다.
  - `before` 와 같이 사용하면 `before`의 첫번째 글자에 적용된다.
```css
.lorem::first-letter{
    color : red;
    font-size : 38px;
}
```
- `first-line` 첫번째 줄에 대한 스타일링을 적용한다.
```css
.lorem::first-line{
    color : red;
    font-size : 38px;
}
```
- `selection` 선택 영역 (드래그) 에 적용되는 스타일링이다.
```css
.lorem::selection{
    background-color: red;
    color : white;
}
```

### 선택자 결합
- `하위` 2가지 선택자를 결합하여 사용한다.
  - 스페이싱을 통해 하위 요소를 구분한다.
  - 아래 예시는 ul 태그에 하위 li 중 마지막 요소를 선택한다.
```css
ul li:last-of-type {
    color : red;
}
```
- `자식` `>` 를 통해 자식을 선택할 수 있다.
  - 범위를 좁혀 유용하게 사용 가능하다. 
```css
#list > li:last-of-type{
    color : red;
}
```
- `형제 선택자` 
  - 일반 형제 선택자 결합 (~)은 같은 부모를 가진 형제들을 선택한다.
    - `code ~ p` 라고 할 때 `code` 아래에 있는 요소만 고를 수 있다.
    - 앞에 있는 셀럭터를 기준으로 뒤에 요소가 있을 때 그 요소의 스타일이 적용된다.
  - 인접 형제 선택자 결합 (+) 바로 인접한 형제를 선택하여 스타일을 적용한다.

```css
code ~ p {
    color : red;
}

code + selector {
    background-color: blue;
}

```
- `그룹화` 선택자가 2가지 이상이지만, 동일한 스타일을 적용할 때 사용한다.
```css
p, span, code {
    color : purple;
}
```

### 범용 선택자
- `*` 요소를 선택하면 모든 요소들에 스타일이 적용된다.
    - CSS 최상단에 쓰는 것이 좋다.
    - `#red` 는 원래 `*.red` 형식이다.
- `p + *` 를 선택하면 `p` 태그 뒤의 모든 요소에 대한 스타일을 정의한다.
```css
* {
    color : red;
}
p.red {
    color : red;
}
p + * {
    color : red;
}
```
### 상속 제어하기
- `initial` 값을 사용하면 부모에 스타일을 상속받지 않는다.
- 특정 스타일이 아닌 모든 스타일에 대한 상속을 거부하려면 `all : initial`을 사용하면된다.
```css
div {
    border : 1px solid silver;
    margin : 2px;
}
.parent {
    color : blue;
}
.child2 {
    color : initial;
}
.child2 {
    all : initial;
}
```
- `inherit` 무조건 상속을 받으라 지정을 할 때 사용된다.
- `all : inherit` 을 사용하면 모든 속성을 부모로 부터 상속받는다.
```css
.parent2 * {
    color : inherit;
}
```
- `unset`
  - 부모로 부터 상속받을 값이 있을 때 : inherit
  - 부모로 부터 상속받을 값이 없을 때 : initial
```css
.parent2 .child1 {
    all : unset;
}
```

### 우선 순위
1. 선언된 곳
    - 위에서 부터 아래로 읽어오면서 늦게 선언된 곳이 적용됨.
    - 브라우저 정의 < 개발자가 선언 < 사용자가 구성
2. 명시도 (적용범위가 적을수록 명시도가 높은 것)
    - inherited < * <Type(tag) < Class/Attribute/Pseudo Class < id (id가 우선순위가 더 높음) < inline < !important
3. 코드 위치
    - 코드 뒤에 작성한 것이 우선순위가 더 높다.
